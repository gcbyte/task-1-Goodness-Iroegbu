A collection of beginner/intermediate cloud and web development projects, covering static site deployment, EC2 server administration, AWS RDS data warehousing, and serverless functions with AWS Lambda.

## Projects

### 1. [Static Portfolio Website](./Project-1-Static-Portfolio-Website)
A personal developer portfolio site built with vanilla HTML and CSS, featuring a light/dark theme toggle, custom typography (Space Grotesk + JetBrains Mono), and sections for About, Skills, Projects, and Experience.

**Stack:** HTML, CSS, vanilla JS
**Files:** `index.html`, `style.css`

### 2. [Server Commander](./Project-2-Server-Commander)
Provisioning and managing a remote Linux server on AWS EC2 — connecting via SSH, configuring Nginx, and serving a static HTML page ("DecodeLabs") over HTTP.

**Stack:** AWS EC2, Ubuntu, Nginx, SSH

### 3. [Data Warehouse](./Project-3-Data-Warehouse)
Setting up and querying a managed relational database using AWS RDS (MySQL), including connecting via the MySQL CLI and inspecting database structure.

**Stack:** AWS RDS, MySQL

### 4. [Serverless](./Project-4-Serverless)
A simple serverless function (`CostCalculator`) deployed on AWS Lambda using Python, demonstrating event-driven compute without managing servers.

**Stack:** AWS Lambda, Python
