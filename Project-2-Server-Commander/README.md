# Project 2: Deploying a Website on Amazon EC2

## Overview

This project demonstrates how to deploy a static website on an Ubuntu EC2 instance using a web server.

## Objectives

- Launch an EC2 instance
- Connect via SSH
- Install a web server
- Deploy a website
- Access the website using the EC2 public IP

## Technologies Used

- Amazon EC2
- Ubuntu Linux
- SSH
- Nginx
- HTML
- CSS

## Steps Performed

1. Created an EC2 instance.
2. Connected securely using SSH.
3. Installed and configured Nginx.
4. Uploaded website files.
5. Verified the web server was running.
6. Accessed the website using the instance's public IP.

## Outcome

The website was successfully hosted on an Amazon EC2 instance and served through the Nginx web server.
