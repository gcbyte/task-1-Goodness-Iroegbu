# Project 4: Serverless Cost Calculator Using AWS Lambda

## Overview

This project demonstrates the use of AWS Lambda to build a serverless function that performs cost calculations without provisioning or managing servers.

## Objectives

- Create an AWS Lambda function
- Write and deploy Lambda code
- Test the function
- Monitor execution using CloudWatch
- Understand serverless computing

## Technologies Used

- AWS Lambda
- Python
- Amazon CloudWatch
- AWS Management Console

## Steps Performed

1. Created a Lambda function.
2. Configured the runtime environment.
3. Implemented the cost calculation logic.
4. Deployed the function.
5. Created test events.
6. Executed and verified the function.
7. Reviewed execution logs in CloudWatch.

## Outcome

Successfully deployed and tested a serverless application using AWS Lambda.

## Skills Demonstrated

- Serverless Computing
- AWS Lambda
- Function Testing
- CloudWatch Monitoring
- Event-Driven Architecture
