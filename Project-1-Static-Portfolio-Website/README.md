# Project 1: Static Portfolio Website Hosting on Amazon S3

## Overview

This project demonstrates how to host a static website using Amazon S3. A personal portfolio website was uploaded to an S3 bucket, configured for static website hosting, and made publicly accessible.

## Objectives

- Create an Amazon S3 bucket
- Upload static website files
- Configure static website hosting
- Set bucket permissions for public access
- Access the website through the S3 website endpoint

## Technologies Used

- Amazon S3
- HTML
- CSS
- AWS Management Console

## Steps Performed

1. Created an S3 bucket.
2. Uploaded the website files (`index.html`, CSS, images).
3. Enabled Static Website Hosting.
4. Configured bucket permissions for public access.
5. Accessed the website using the S3 website endpoint.

## Outcome

The website was successfully deployed and made accessible over the internet using Amazon S3 Static Website Hosting.
